## Overview

A full-stack personal project built to deepen backend engineering — REST API design, authentication, and automated UI testing — applying software-engineering principles beyond testing. Source is on GitHub at github.com/sontbui/E-commerce-Platform.

## Does Son have backend experience?

Yes. This project is the clearest evidence: Son designed and built a **Spring Boot REST API** with **Spring Security and JWT** authentication over a **MySQL** relational store, with an **Angular** single-page front end (TypeScript · Material UI). It demonstrates real backend engineering — API design, auth, and data modelling — not just testing.

## 3-tier architecture

- **Angular SPA** (TypeScript · Material UI) — the client tier.
- **Spring Boot REST API** (Spring Security · JWT) — the application tier.
- **MySQL** — the relational data store.

## Testing

Selenium exercises the UI end-to-end, giving automated regression across the full stack — Son applying his testing discipline to his own product.
